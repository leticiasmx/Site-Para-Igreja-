## 🚀 Tecnologias

- HTML5
- TailwindCSS
- Font Awesome 6

- História completa (1942-2025)
- Ministério Infantil (Silvana: 99 99841-87454)
- Doações via PIX e transferência
- Botão copiar dados
- WhatsApp flutuante
- Mapa interativo
