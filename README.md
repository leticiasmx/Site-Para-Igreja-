## 📋 Sobre

Site institucional da **Primeira Igreja Batista de Timon (MA)** com história, programação, ministério infantil e doações.

## 🚀 Tecnologias

- HTML5
- TailwindCSS
- Font Awesome 6

## 📱 Funcionalidades

- Design responsivo
- Menu mobile
- História completa (1942-2025)
- Ministério Infantil 
- Doações via PIX e transferência
- Botão copiar dados
- WhatsApp flutuante
- Mapa interativo
